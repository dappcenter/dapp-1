# FE :: 前端项目


## 开发环境搭建


#### 安装依赖
    $ npm install

#### 启动开发环境
    $ npm run dev

##### 打包编译
    $ npm run build

#### 测试

    $ npm run unit  # run unit tests
    $ npm run e2e   # run e2e tests
    $ npm test      # run all tests
